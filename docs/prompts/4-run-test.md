# テスト実行プロンプト

## 目的

実装した server.js を server.test.js でテストし、すべてのテストがパスすることを確認します。

## いつ使う

- server.js の実装後
- server.js を修正した後

## 指示

以下のコマンドでテストを実行し、結果を確認してください。

### テスト実行コマンド

```bash
# 全テスト実行
npm test

# ウォッチモードで実行
npm test -- --watch

# 特定のテストのみ実行
npm test -- server.test.js -t "テスト名"
```

### 確認事項

1. **すべてのテストがパスすること**
   - 失敗がある場合は、server.js を修正

2. **カバレッジの確認**（オプション）
   ```bash
   npm test -- --coverage
   ```

### テスト失敗時の対応

1. 失敗したテストを特定
2. 期待される動作と実際の動作を比較
3. server.js を修正
4. 再度テスト実行

## 実行方法

```bash
npm test
```

すべてのテストがパスしたら、次のステップ（OpenAPI仕様書生成）に進んでください。
