# mablテスト作成プロンプト

## 目的

mabl で API テストを作成します。

## いつ使う

- 新しい API テストを mabl で作成したい時
- API の正常系・異常系テストを追加したい時
- CI/CD パイプラインにテストを組み込みたい時

## テスト内容

以下のAPIテストを作成してください：

### 1. セットアップ
- ログイン（JWTトークンを取得し、以降のAPIアクセス時に使用）
- デモ環境リセット
- シードデータ作成

### 2. 成功パターン
- 注文作成（item: "成功商品"）
- 支払い処理（created → paid）
- 発送処理（paid → shipped）

### 3. 失敗パターン
- 注文作成（item: "失敗商品"）
- 発送処理を支払い前に実行（paidをスキップ → エラー）

## アサーション

- ステータスコードの確認
- レスポンスボディの構造確認
- ステート遷移の確認

## 実行方法

mabl MCP サーバー経由で、このプロンプトの内容に従ってテストを作成してください。

```
docs/prompts/7-create-mabl-test.md の指示に従って、mabl APIテストを作成してください。
```
